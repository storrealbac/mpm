mpm

Minecraft Plugin Manager

Install

Linux / macOS:

curl -fsSL https://storrealbac.github.io/mpm/install.sh | bash

Windows:

powershell -c "irm https://storrealbac.github.io/mpm/install.ps1 | iex"

Usage

mpm init                    # Initialize new server
mpm install <plugin>        # Install plugin
mpm update                  # Update all plugins
mpm list                    # List installed plugins

Supported Servers

Paper, Purpur, Folia, Spigot, Bukkit, Sponge, Velocity, Waterfall
